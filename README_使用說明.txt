病人日常動作教學｜手機版網頁APP 使用說明

內容：
- 10個分類、46張圖片。
- 可用手機瀏覽，按分類進入後可看步驟圖。
- 點圖片可放大，並可用上一張／下一張切換。
- 右上「字放大」可切換較大字體。

使用方式：
1. 先解壓縮此ZIP檔。
2. 開啟資料夾中的 index.html。
3. 若要讓病人更方便使用，建議上傳整個資料夾到網站空間、學校伺服器、GitHub Pages或其他靜態網頁空間。
4. 若以網站方式開啟，可在手機瀏覽器使用「加入主畫面」做成類似APP的圖示。

安全提醒：
本APP為圖片輔助教學，不能取代醫師、物理治療師、職能治療師或護理人員指導。
若病人出現疼痛、頭暈、喘、失衡、跌倒風險或其他不適，請立即停止並尋求專業協助。
